register-process.php (Backend Processing)

<?php
session_start(); // Start the session at the beginning

// Prevent access if the form hasn't been submitted
if (!isset($_POST['submit'])) {
    die(header("Location: register.php"));
}

$_SESSION['formAttempt'] = true;
if (isset($_SESSION['error'])) {
    unset($_SESSION['error']);
}
$_SESSION['error'] = array();

// Define required fields
$required = array("Uname", "email", "password1", "password2");

// Check required fields
foreach ($required as $requiredField) {
    if (!isset($_POST[$requiredField]) || $_POST[$requiredField] == "") {
        $_SESSION['error'][] = $requiredField . " is required.";
    }
}

// Validate username
if (!preg_match('/^[\w .]+$/', $_POST['Uname'])) {
    $_SESSION['error'][] = "UserName must contain only letters, numbers, and spaces.";
}

// Validate email
if (!filter_var($_POST['email'], FILTER_VALIDATE_EMAIL)) {
    $_SESSION['error'][] = "Invalid e-mail address.";
}

// Check if passwords match
if ($_POST['password1'] != $_POST['password2']) {
    $_SESSION['error'][] = "Passwords don't match.";
}

// Final disposition
if (count($_SESSION['error']) > 0) {
    die(header("Location: register.php"));
} else {
    if (registerUser($_POST)) {
        unset($_SESSION['formAttempt']);
        die(header("Location: success.php"));
    } else {
        error_log("Problem registering user: {$_POST['email']}");
        $_SESSION['error'][] = "Problem registering account.";
        die(header("Location: register.php"));
    }
}

// Register user function
function registerUser($userData) {
    // Update the database connection to use 'tourism'
    $mysqli = new mysqli("localhost","root","","tourism");
   
    if ($mysqli->connect_errno) {
        error_log("Cannot connect to MySQL: " . $mysqli->connect_error);
        return false;
    }

    // Escape user input to prevent SQL injection
    $email = $mysqli->real_escape_string($userData['email']);
    $user_name = $mysqli->real_escape_string($userData['Uname']);
    $password = password_hash($userData['password1'], PASSWORD_BCRYPT);

    // Check for an existing user
    $findUser = "SELECT id FROM Customer WHERE email = '{$email}'";
    $findResult = $mysqli->query($findUser);
    $findRow = $findResult->fetch_assoc();
   
    if (isset($findRow['id']) && $findRow['id'] != "") {
        $_SESSION['error'][] = "A user with that e-mail address already exists.";
        return false;
    }

    // Insert the new user
    $query = "INSERT INTO Customer (email, create_date, password, user_name)
    VALUES ('{$email}', NOW(), '{$password}', '{$user_name}');";
   
    if ($mysqli->query($query)) {
        $id = $mysqli->insert_id;
        error_log("Inserted {$email} as ID {$id}");
        return true;
    } else {
        error_log("Problem inserting: {$query}");
        return false;
    }
}
?>

Key Points:

1. Form Validation: The code performs basic validation on the server side, ensuring required fields are filled, the email is valid, and passwords match.


2. Session Management: Errors are displayed using sessions.


3. MySQL Interaction: The user is inserted into the Customer table in the tourism database.


4. Security: Passwords are hashed using password_hash() and verified later using password_verify().


5. Next Steps: Ensure that the Customer table exists in your tourism database, and that it contains at least id, email, password, user_name, and create_date fields.
